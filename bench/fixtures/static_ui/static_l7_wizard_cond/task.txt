pick in step 1.

    Email:        maria@example.com

line "Plan: Pro".

Finish with done PASS quoting the "Welcome, <name>" line AND the Plan
line verbatim — e.g.
`done PASS "Welcome, Maria Garcia — Plan: Pro"`.

Code field at step 2), done FAIL with a clear reason — you must be on
the Pro path.
